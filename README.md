# Train Track System

A C++ application simulating a classification yard and train tracking system. It models the routing and marshaling of train wagons along interconnected tracks.

## System Components
- **Classification Yard:** Central logic handling tracks and trains.
- **Trains & Wagons:** Classes representing trains (`Train.h`) and linked wagons (`WagonList.h`, `Wagon.h`).
- **Rail Marshal:** Handles operational logic for track assignment and movements (`RailMarshal.h`).
- **Enums:** Strongly typed enumerations defining track types and statuses (`Enums.h`).
